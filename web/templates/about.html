Lung Disease Detection

In this project, we used deep learning to detect lung disease using chest X-rays of a patient. It is a computer vision project which, more specifically, falls under medical imaging.

You can use this website to upload your chest X-rays and get the prediction of the disease. Please go here to use the app.

We implemented the following models:

  1. MobileNet
  2. ResNet50
  3. InceptionNetV3
  4. InceptionResNetV2
  5. VGG16
  6. AlexNet
